## API Documentation


`GET /`: Return all the axis to the cartesian plan to day ZERO

#### Query example:

```
    "airports": 3,
    "clouds": 4,
    "rows": 10,
    "columns": 10
```


`GET /nextday`: Return all the axis to the cartesian plan to the nextday
